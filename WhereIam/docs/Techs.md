# Techs

This project uses the following main technologies:

- **React Native**: Main framework for cross-platform mobile app development.
- **Expo**: Platform for developing, building, and deploying React Native apps.
- **TypeScript**: Static typing for JavaScript.
- **Expo Router**: File-based routing for navigation.
- **React Navigation**: Navigation between screens.
- **Restyle**: Design system and theming for React Native.
- **Reanimated**: Smooth, native animations.
- **VS Code**: Recommended code editor.
