# Libraries

Main libraries used in this project:

- **expo**: Core Expo SDK.
- **expo-router**: File-based navigation.
- **expo-location**: Access to device location.
- **expo-image**: Image optimization and loading.
- **expo-status-bar**, **expo-splash-screen**, **expo-constants**, **expo-haptics**, **expo-web-browser**, **expo-localization**, **expo-system-ui**, **expo-font**: Various Expo utilities.
- **@react-navigation/native**, **@react-navigation/bottom-tabs**, **@react-navigation/elements**: Navigation.
- **react-native-heroicons**: SVG icons.
- **react-native-maps**: Maps integration.
- **react-native-reanimated**: Advanced animations.
- **@shopify/restyle**: Design system and theming.
- **react-native-safe-area-context**, **react-native-screens**, **react-native-svg**: UI utilities.
- **typescript**, **eslint**: Development and code quality tools.

See `package.json` for the full list and versions.
